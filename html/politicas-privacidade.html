Política de Privacidade

Esta Política de Privacidade descreve como o site do Colégio Futura coleta, utiliza e protege as informações fornecidas pelos usuários através do formulário de matrícula.

1. Dados Coletados

Quando você preenche o formulário de matrícula, coletamos os seguintes dados pessoais:

  • Nome completo
  • E-mail
  • Telefone
  • Data de nascimento

2. Finalidade da Coleta de Dados

Esses dados são coletados com o único objetivo de processar sua solicitação de matrícula e facilitar o contato da escola com você.

3. Como os Dados São Utilizados

As informações fornecidas são enviadas diretamente para o WhatsApp da escola, onde nossa equipe poderá analisá-las e entrar em contato para prosseguir com o processo de matrícula.

4. Compartilhamento de Dados

As informações coletadas não serão compartilhadas com terceiros e serão usadas exclusivamente pela equipe do Colégio Futura.

5. Segurança dos Dados

O Colégio Futura se compromete a proteger suas informações pessoais e manter a confidencialidade dos dados compartilhados através deste site.

6. Direitos dos Usuários

Como os dados fornecidos são enviados diretamente para o WhatsApp da escola, não mantemos um banco de dados com essas informações. Caso deseje excluir ou corrigir qualquer informação que tenha sido fornecida, entre em contato conosco através de nossos canais de atendimento.

7. Alterações na Política de Privacidade

Essa política poderá ser atualizada periodicamente. Recomendamos que você revise esta página para estar sempre ciente de nossas práticas.
